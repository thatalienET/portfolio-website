I started this project as a challenge to see how I could bring an animation to life by using Flieds in Cinema 4D


Once I completed the Initial animation I was excited to add flare to it by incorporating some hand drawn animations which enhanse the movements and just look cool! I droped the keyboard animations into Photoshop and drew on top of the footage - creating shapes which enhanse the movements and add in some fun!

I designed my own lettering in Illustrator using colours and overlapping shapes to create a fun custom font
